# SmartAcademy - Plataforma Educativa

Este proyecto simula una plataforma educativa con páginas para iniciar sesión, registrarse, ver cursos y navegar entre ellas.

## Estructura

- `index.html`: Página de inicio
- `login.html`: Inicio de sesión
- `register.html`: Registro de usuarios
- `dashboard.html`: Panel del estudiante
- `styles.css`: Estilos
- `script.js`: Interacciones con formularios

## Tecnologías

- HTML5
- CSS3
- JavaScript
